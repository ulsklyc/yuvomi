5. Speicherung im lokalen Browser-Speicher (LocalStorage)

Damit Ihre Anzeige-Präferenzen über Seitenaufrufe hinweg erhalten bleiben, speichert die Website folgende Werte im localStorage Ihres Browsers:

Schlüssel Inhalt Zweck Speicherdauer

oikos-lang z. B. de / en Beibehaltung der gewählten Sprache bis zur Löschung durch Sie

Rechtsgrundlage: § 25 Abs. 2 Nr. 2 TDDDG. Die Speicherung dieser Präferenzen ist unbedingt erforderlich, damit der von Ihnen ausdrücklich gewünschte Telemediendienst — die Darstellung der Website in der von Ihnen ausgewählten Sprache und Farbgebung — bereitgestellt werden kann. Eine Einwilligung ist daher nicht erforderlich. Eine Übermittlung dieser Werte an Dritte oder an Server findet nicht statt.

So löschen Sie diese Werte: Über die Browser-Einstellungen unter „Website-Daten löschen" / „Cookies und andere Website-Daten" oder über die Entwicklerwerkzeuge (Application → Local Storage → yuvomi.cloud).
